# large-scale-electricity-load-forecasting

This project focuses on multi-horizon electricity load forecasting using statistical and deep learning models. The study evaluates model performance across different electricity-demand datasets, forecast horizons, and model architectures, with particular attention to the role of calendar-based exogenous variables.

## Project Overview

Electricity load forecasting is an important task in modern power and energy systems. Reliable forecasts can support grid planning, demand management, renewable integration, and data-driven decision-making.

This project compares classical statistical models and deep learning architectures for electricity demand forecasting.

## Models

The project includes or evaluates the following forecasting approaches:

- ARIMA
- N-BEATSx
- N-HiTS
- Temporal Fusion Transformer (TFT)

## Main Focus

- Multi-horizon electricity load forecasting
- Time-series forecasting for energy systems
- Deep learning for electricity-demand modeling
- Calendar-based exogenous variables
- Forecast horizon analysis
- Model comparison across datasets

## Datasets

The study uses large-scale electricity-demand datasets, including:

- ECL dataset
- PJM electricity load dataset
