# Project Summary

This project implements a NeuralForecast-based experimental pipeline for multi-horizon electricity load forecasting.

The pipeline supports N-HiTS, N-BEATS, and N-BEATSx models, with configurable forecast horizons, input window lengths, exogenous-variable settings, fixed and rolling-origin evaluation, multi-seed experiments, metrics export, reporting, and computational cost analysis.

The project is based on my MSc thesis in Machine Learning, Systems and Control at Lund University.
